  • encode(to:)

    Encodes this record. NOTE! Only the values set by calling setValue(_:,forField:) are encoded; values that were decoded from Salesforce are not encoded here. That’s by design since an insert or update operation requires only new values, and should not include fields that can’t be updated, e.g. Id or CreatedDate
