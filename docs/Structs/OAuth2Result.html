    Initializer

    Declaration

    internal init(accessToken: String, instanceURL: URL, identityURL: URL, refreshToken: String?)

    Parameters

    accessToken

    Salesforce access token (session ID).
    instanceURL

    Salesforce instance to use for subsequent, authenticated requests.
    identityURL

    Salesforce identity URL unique to the current user. See: https://help.salesforce.com/articleView?id=remoteaccess_using_openid.htm&type=0
    refreshToken

    Optional value for refresh token; since the refresh token flow doesn’t return the refresh token itself, including the refresh token as an argument will preserve it for the next refresh request.
